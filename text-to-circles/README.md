# Текст в круг — генератор скриншотов

Web-приложение для размещения текста с формулами внутри круга 470×470px с последующим экспортом в PNG.

## Как запустить

1. Откройте `index.html` в браузере (Chrome, Firefox, Edge).
2. Требуется интернет-соединение для загрузки библиотек (KaTeX, JSZip, FileSaver).

Или запустите локальный сервер (рекомендуется для корректной работы):

```bash
# Python
python -m http.server 8080

# Node.js (npx)
npx serve .
```

И откройте `http://localhost:8080`.

## Использование

1. **Введите текст** в поле слева. Формулы записываются в LaTeX:
   - Строчные: `$E = mc^2$`
   - Блочные: `$$\int_0^1 x\,dx$$`
   - Unicode тоже поддерживается: `x² + √π ≈ 3`

2. **Настройте параметры** (размер шрифта, цвета и т.д.).

3. **Область вырезания** (опционально) — задайте SVG-путь или нарисуйте на холсте:
   - ЛКМ — добавить точку полигона
   - ПКМ — закрыть фигуру
   - Ctrl+Z — отменить

4. Нажмите **▶ Сгенерировать** (или Ctrl+Enter).

5. В правой панели появится предпросмотр. Под каждой страницей — кнопка **Скачать PNG**.

6. Кнопка **📦 Скачать все (ZIP)** — скачать все страницы архивом.

## Структура файлов

```
text-to-circles/
├── index.html           # Главная страница
├── css/
│   └── styles.css       # Стили
├── js/
│   ├── app.js           # Координация модулей
│   ├── textParser.js    # Парсер текста + LaTeX
│   ├── circleRenderer.js # Рендеринг в круг
│   ├── pathEditor.js    # Редактор области вырезания
│   └── exportManager.js # Экспорт PNG/ZIP
└── README.md
```

## Технологии

- **KaTeX** — рендеринг LaTeX-формул
- **Canvas API** — графика и экспорт
- **JSZip + FileSaver.js** — создание ZIP-архивов
- **SVG Path2D** — область вырезания
